# Shinil Elementary Name Matching

[![Vercel Deployment](https://img.shields.io/badge/vercel-deployed-brightgreen)](https://shinil-elementary-name-matching.vercel.app)

신일교회 초등부 교사들이 학생들의 이름과 얼굴을 효과적으로 익힐 수 있는 웹 애플리케이션입니다.

![신일교회 초등부 앱](https://user-images.githubusercontent.com/assets/sample.png)

## 📚 주요 기능

### 🔍 목록 모드
- 전체 학생 목록을 한눈에 확인
- 학년별 필터링 (4, 5, 6학년) 기능
- 학생 정보 및 얼굴 사진 제공

### 📖 외우기 모드
- 카드형 UI로 학생 얼굴 확인
- 카드를 클릭하면 뒤집혀 이름 확인 가능
- 랜덤 순서 섞기 기능

### ✏️ 퀴즈 모드
- 학생 사진을 보고 이름 맞추기 (객관식)
- 자동으로 다음 문제로 넘어가는 빠른 학습
- 퀴즈 결과와 틀린 문제 한눈에 확인

### 🎓 시험 모드
- 학생 사진을 보고 이름 직접 입력
- 실시간 정답 확인
- 시험 결과와 틀린 문제 한눈에 확인

## 🌈 주요 특징

- **직관적인 UI/UX**: 교사들이 쉽게 사용할 수 있는 인터페이스
- **반응형 디자인**: 모든 디바이스에서 최적화된 경험
- **학습 효율성**: 다양한 학습 방식을 통한 효과적인 이름 암기
- **학년별 필터링**: 담당 학년에 맞춰 선택적 학습 가능

## 💻 기술 스택

- **Frontend**: React, TypeScript
- **스타일링**: Styled-Components
- **라우팅**: React Router
- **배포**: Vercel

## 🚀 시작하기

1. 로그인 화면에서 교사용 비밀번호 입력
2. 네비게이션 바에서 원하는 학습 모드 선택
3. 필요에 따라 학년 필터링 적용
4. 학습 진행

## 📱 접근성

- **데스크톱**: 모든 기능 최적화
- **태블릿/모바일**: 반응형 디자인으로 모바일에서도 편리하게 사용 가능
