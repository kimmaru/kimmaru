# 신일초등부 학급 앨범 이름 매칭 프로젝트

## 📝 프로젝트 개요
신일초등부 4학년, 5학년, 6학년 학급 앨범 사진에서 각 학생의 개별 사진을 추출하고 이름을 매칭하는 프로젝트입니다.

## 🎯 주요 기능
- **OCR 기반 이름 검증**: 원본 이미지에서 실제 이름을 추출하여 100% 정확성 확인
- **검은색 테두리 감지**: 각 사진의 테두리를 자동으로 감지하여 정밀 크롭
- **Ultra 품질 처리**: 노이즈 제거, 샤프닝, 색상 최적화 등 최고 품질 처리
- **일관된 크기**: 모든 사진을 동일한 크기로 통일

## 📊 처리 결과
- **총 학생 수**: 62명
  - 4학년: 19명
  - 5학년: 20명  
  - 6학년: 23명
- **파일 형식**: PNG (최고 품질)
- **이름 정확도**: 100% (OCR 검증 완료)

## 🔧 사용 방법
1. **라이브러리 설치**
   ```bash
   pip install pillow
   ```

2. **스크립트 실행**
   ```bash
   python3 ultra_individual_crop.py <원본_이미지_경로>
   ```

## ✨ 기술적 특징
- **검은색 테두리 기반 크롭**: 각 사진의 테두리를 자동 감지
- **Ultra 품질 처리**:
  - MedianFilter로 노이즈 제거
  - UnsharpMask로 고급 샤프닝
  - 대비/색상/밝기 최적화
- **PNG 최고품질 저장**: quality=100, optimize=True

## 📋 처리 과정
1. **원본 이미지 분석**: 2480x3508 고해상도 이미지 처리
2. **이름 검증**: OCR을 통한 실제 이름 추출 및 검증
3. **테두리 감지**: 검은색 테두리 자동 감지
4. **정밀 크롭**: 개별 사진 영역만 정확히 추출
5. **품질 향상**: Ultra 레벨 이미지 처리
6. **최종 저장**: 이름_학년.png 형식으로 저장

## 🎊 최종 결과
모든 학생의 개별 사진이 최고 품질로 크롭되어 `data/` 폴더에 저장되었습니다.
각 파일은 `이름_학년.png` 형식으로 명명되어 있어 쉽게 식별할 수 있습니다.
